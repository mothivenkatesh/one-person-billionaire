# GTM Analytics Skills (40 skills)

Enterprise-grade GTM analytics workflows across **Marketing / Sales / RevOps / CX / Systems & Data**. These skills are calibrated for the second-stage of the curriculum â€” when you scale past $1M ARR and need real revenue ops discipline.

## When to use these skills

These skills assume:
- A connected CRM (Salesforce / HubSpot)
- Marketing automation (Marketo / HubSpot Marketing)
- A data warehouse (Snowflake / BigQuery / Postgres)
- Multiple reps / campaigns / segments to analyze

**Sized for $1M+ ARR teams with sales + marketing + CS functions.**

For solo operators below $1M ARR, the parent curriculum's skills (`wedge-finder`, `cold-outbound-drafter`, `retention-cohort-analyzer`, etc.) are calibrated to your stage.

## Where this fits in the journey

- **Year 1-2:** $0 â†’ $100K MRR â€” use parent curriculum skills (find wedge, ship MVP, first 100 customers)
- **Year 2-3:** $100K â†’ $1M MRR â€” start using these GTM analytics skills as you build sales + marketing functions
