## 3. Crowd check (weekly)

- Did I talk to ≥ 3 prospects/customers this week? Y/N
- Did I hear the same complaint twice? Y/N → if yes: ____________
- Is the dream outcome statement still accurate? Y/N → if no: rewrite

---

## 4. Bonus stack health

- [ ] Each bonus still addresses a real objection (not fluff)
- [ ] Total bonus value still 5-10× price
- [ ] No new objections came up this week that don't have a bonus addressing them

If new objection surfaced → which one? → which new bonus addresses it?

> New objection:
> New bonus to add:

---

## 5. Pricing pulse

- Last week's quoted price: $______
- Anyone said "too expensive"? Y/N (if no — raise next quote)
- Anyone said "what a steal"? Y/N (if yes — raise meaningfully)
- Close rate this week: ____% (vs ____% last week)

---

## 6. Guarantee health

- Did anyone invoke the guarantee this week? Y/N
- If yes: was it a fair invocation or a bad-fit customer?
- Should the guarantee conditions tighten/loosen?

---

## 7. Number of "stupid no" reactions this week

How many prospects said "feel stupid saying no" or equivalent? Track this. If 0 for 2+ weeks → offer is too weak; revisit Lesson 08A.
